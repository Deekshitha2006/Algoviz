How to Use AlgoViz

Your AI lab companion — built for students who want to understand, not just memorise

01

Pick an Algorithm

From the sidebar, choose BFS, DFS, Best-FS, UCS, or IDDFS. The complexity card at the bottom updates instantly.

02

Grid Mode

Type a grid size → New Grid. Click → Start (green). Click → Goal (red). More clicks → Walls. Then ▶ Run.

03

Custom Graph — Type

Switch to ⬡ Custom Graph → click 📝 Type Edges → enter edges like A-B one per line → Build Graph.

04

Custom Graph — Draw

Use ＋ Node to click and place nodes on the canvas. Use ⟶ Edge to connect two nodes. Drag nodes to reposition.

05

Set Start & Goal

Right-click any node on the custom graph to open a menu. Choose Set as Start or Set as Goal. Then press ▶ Run.

06

Pause & Step

Hit ⏸ Pause to freeze the animation. Then ⏭ Step moves one action at a time — best way to learn what's happening.

5 Algorithms — Plain English

BFS

Uses a Queue (FIFO). Explores level by level. Always finds the shortest path.

✅ Shortest path guaranteed
DFS

Uses a Stack (LIFO). Dives deep first. Not always the shortest path.

⚠️ Not always shortest path
Best-FS

Uses heuristic distance to goal. Greedy — picks what looks closest. Fast, not always optimal.

⚡ Fast, not always optimal
UCS

Expands cheapest cumulative cost first. Optimal even with different edge weights.

✅ Optimal for weighted graphs
IDDFS

DFS with depth limit 0,1,2... Gets BFS-optimal result with very low memory.

✅ Optimal + low memory
